How It Works

The Game

European marble solitaire has 37 holes arranged in a cross pattern. Start with 36 marbles (center empty). Jump a marble over an adjacent one to remove it. Goal: end with just 1 marble. Most games end stuck with many marbles — there are billions of possible games.

The Neural Network

A small "brain" (convolutional neural network with ~75,000 parameters) looks at the board and predicts:

  • Policy: which move to make (probability for each possible jump)
  • Value: how good is this position (score from -1 to +1)

MCTS (Monte Carlo Tree Search)

Before each move, the model "imagines" hundreds of possible futures. It combines the neural network's intuition with simulated lookahead to make better decisions — like a chess player thinking ahead.

How It Learns (AlphaZero Loop)

  1. Play many games using MCTS (self-play)
  2. Record which moves were tried and how each game ended
  3. Train the neural network on these experiences
  4. Repeat — each generation plays better than the last

What You're Seeing

The 5 generations show the model improving from random (Gen 1) to skilled (Gen 5). Gen 1 has never seen a solitaire board. Gen 5 has played thousands of games and discovered strategies on its own.
